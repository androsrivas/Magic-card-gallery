# MAGIC CARD GALLERY

Aplicación web para explorar cartas de Magic: The Gathering construida con Vite y React.

## Tecnologías
- Javascript
- React
- Vite
- HTML
- CSS

## Descripción
Magic Card Gallery es un proyecto frontend que consume datos de cartas para mostrar una galería interactiva. El foco principal del proyecto es practicar el desarrollo de interfaces con React y gestionar datos desde una API (o mock).

## Funcionalidades implementadas
- Estructura de proyecto con React + Vite
- Interfaz de usuario para mostrar cartas
- Estilos básicos con CSS
- Componentes reutilizables

> Proyecto en desarrollo con foco en frontend y estructura de componentes.

## Cómo ejecutar el proyecto
1. Clonar este repositorio
```bash
git clone https://github.com/androsrivas/Magic-card-gallery.git
```

2. Insatalar dependencias
```bash
npm install
```
o
```bash
pnpm install
```

3. Iniciar aplicación
```bash
npm run dev
```
o

```bash
pnpm run dev
```

## Estructura principal
- `/src`: código fuente de la aplicación.
- `index.html`: punto de entrada de la app.
- `vite.config.js`: configuración de Vite.

## Estado del proyecto
Este proyecto está en desarrollo y sirve como demostración de conocimientos de React y configuración de proyectos modernos con Vite.
